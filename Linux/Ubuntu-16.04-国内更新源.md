# #Ubuntu-16.04-国内更新源

1. 在修改source.list前，最好先备份一份

```cmd
sudo cp /etc/apt/sources.list /etc/apt/sources.list.old
```

2. 执行命令打开source.list文件

```cmd
sudo vim /etc/apt/source.list
 ```
 
3. 复制源

把下面的任意一下复制到source.list中去，并覆盖原来的文件内容。
